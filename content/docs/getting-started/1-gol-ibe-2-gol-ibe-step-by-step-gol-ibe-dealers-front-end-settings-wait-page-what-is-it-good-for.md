# Wait page — what is it good for?

<!-- tags: wait page, loading page, search results, city image, front-end, user experience -->

This feature allows you to show an image of the arrival city selected by the client.

**1. Prepare Your Images:**
- Create a folder on your system containing images of various cities.
- Name each image using the IATA code of the corresponding city (e.g., the image for Paris should be named PAR).

**2. Link the Folder:**
- Insert the link to this folder in the Wait Page section.
- Use the following format for the link: `/files/7cbNwISbHY6SvzHl51LT`

**3. Save Changes.**

When your customers search for a flight, the corresponding image from your folder for the selected arrival city will be displayed.
